# Raspberry-Pi

This repository consists of Python programs which I coded while learning to access various hardwares connected to a Raspberry Pi.

* 7-Segment Display
* Active/Passive Buzzer
* Automated Rubik's Cube Solver (ARCS)
* Binary Clock
* DC Motor
* LCD 16x2
* LED
* Luma LED Matrix
* Morse Code
* Push Button
* Sense HAT
* Servo Motor
* Stepper Motor
* Ultrasonic Sensor
